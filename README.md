## 📖 Sobre o Projeto

Este projeto consiste na personalização da aplicação **MeuCurriculo**, desenvolvida com Blazor. O objetivo foi transformar o modelo apresentado em aula em um currículo digital mais profissional, contendo informações acadêmicas, habilidades e experiências relevantes.

---

## 🛠️ Tecnologia Utilizada

* Blazor
* C#
* .NET
* HTML
* CSS

---

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone <url-do-repositorio>
```

2. Acesse a pasta do projeto:

```bash
cd MeuCurriculo
```

3. Execute a aplicação:

```bash
dotnet run
```

4. Abra o navegador no endereço informado pelo terminal.
